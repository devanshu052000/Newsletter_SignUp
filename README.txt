Newsletter Signup App

Created a web app using which users can sign up our newsletter.

Motivation

Developed it while completing the course "The Complete 2022 Web Development Bootcamp" on Udemy. Aim was to learn how APIs work and how to make HTTP GET and POST requests to API using NodeJS.

Tech/Framework used

HTML, CSS, Bootstrap, ExpressJS, NodeJS, MailChimp API.

Installation

Install all the dependencies by typing the command "npm install" inside command line.

Tests

Enter your first and last name and then enter your email. Then press 'Sign me Up!' button.
